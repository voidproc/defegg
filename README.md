# defegg

Siv3D Game Jam 第一回 テーマ「たまご」の投稿作品です．

マウスでたまごちゃんを敵から護るんだ．

## 製作者
<https://twitter.com/voidproc>

## 遊び方

* マウスに追従するシールドでたまごを敵から護ろう．
* ピンチになったら右クリックでボンバーだ．
